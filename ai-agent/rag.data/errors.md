# Common Errors and Solutions

## Unauthorized Access
Reason:
- JWT token missing or invalid

Solution:
- Login again

---

## Message Sending Failed
Reason:
- Server issue
- Invalid receiver
- Network problem

Solution:
- Check connection
- Retry sending message

---

## Server Not Responding
Reason:
- Backend down
- API issue

Solution:
- Refresh page
- Try again later
