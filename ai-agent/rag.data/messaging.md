# Messaging System

Users can communicate with each other through messages.

## How it works
1. User selects another user
2. Types a message
3. Sends the message
4. Message is stored and delivered

## Common Issues

### Message not sending
Possible reasons:
- Receiver does not exist
- Network issue
- Server error

### Messages not दिखाई दे रहे
- Chat not refreshed
- Backend not responding

## Tips
- Ensure internet connection is stable
- Try refreshing the chat
