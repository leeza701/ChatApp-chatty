# Features

## Authentication
- Users can sign up and log in securely
- JWT is used for session management

## Messaging
- Users can send and receive messages
- Messages are stored in the database
- Conversations are real-time (if socket used)

## User Profiles
- Each user has a profile
- Includes name and profile picture

## AI Assistant
- Users can ask questions about the app
- AI helps with issues and feature explanations
